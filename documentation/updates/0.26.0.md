# Arcane Council
- Introduced new Arcane Council international organization called
- The Arcane Council becomes available after international incidents involving Spellcasters
- Incidents start after 10 or more countries have access to magic and mutual interest in a region
- Aside from preventing these incidents the council also offers a platform for further diplomatic interaction
- Founding the council creates a building in your capital that provides prestige
- Added 'Foreign Magic, Domestic Consequences' incident event
- Added 'Blight of the Outsider' incident event
- Added 'Research Cooperation' action
- Added 'Curtail Battlefield Magic' action
- Added 'Further Curtail Battlefield Magic' action
